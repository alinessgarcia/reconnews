# Design System — Lumen Daily

## Colors
| Token | Hex | Usage |
|-------|-----|-------|
| `primary` | `#3B82F6` | Buttons, active tabs, links |
| `primary-light` | `#60A5FA` | Hover states, accents |
| `primary-dark` | `#2563EB` | Pressed states, gradients |
| `accent-warm` | `#D97706` | Recipe/nutrition highlights |
| `accent-green` | `#2bee7c` | Success, health indicators |
| `surface` | `#F5F5F5` | Card backgrounds, pills |
| `surface-blue` | `#DBEAFE` | Workout library bg |
| `text-primary` | `#1F2937` | Headings, body text |
| `text-secondary` | `#6B7280` | Captions, metadata |
| `text-muted` | `#9CA3AF` | Inactive nav, placeholders |

## Typography
- **Font**: Lexend (Google Fonts)
- **Icons**: Material Symbols Outlined

## Spacing
- Border radius: `0.75rem` for cards
- Header height: `64px`
- Bottom nav height: `68px`
- Content padding: `20px` horizontal
